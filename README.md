**AI Article Generator (Make.com + Gemini)**

A modern, lightweight web application that generates professional 500-word articles using Google Gemini, powered by a Make.com automation workflow and a clean HTML/CSS/JavaScript frontend.

This project is fully static, requires no backend, and is deployable on GitHub Pages.
